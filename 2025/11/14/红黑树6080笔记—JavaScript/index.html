JavaScript函数声明与函数表达式的区别:

在 JavaScript 中，函数声明（Function Declaration）和函数表达式（Function Expression）是两种定义函数的方式。它们的主要区别在于语法、作用域提升（Hoisting）以及使用场景。

3.4 函数参数

详细讲解一下第二个默认参数

…numbers

三个点 … 的意思是：把所有参数装进一个盒子（数组）里

然后再讲解一下数组的reduce方法

基本概念
reduce 的作用是：把数组中的所有元素，通过某种规则，合并成一个值

参数说明：

  • 累加器（accumulator）：存储每次计算的结果
  • 当前元素（current）：当前遍历到的数组元素
  • 初始值：累加器的起始值

例子1:

执行过程：

初始：total = 0

第1轮：num = 1 → total = 0 + 1 = 1
第2轮：num = 2 → total = 1 + 2 = 3
第3轮：num = 3 → total = 3 + 3 = 6
第4轮：num = 4 → total = 6 + 4 = 10

返回 10

例子2：找最大值

执行过程：

初始：maxNum = 5

第2轮：current = 2 → 2 > 5? 否 → maxNum = 5
第3轮：current = 8 → 8 > 5? 是 → maxNum = 8
第4轮：current = 1 → 1 > 8? 否 → maxNum = 8
第5轮：current = 9 → 9 > 8? 是 → maxNum = 9

返回 9

例子3： 数组扁平化

执行过程：

初始：result = []

第1轮：arr = [1, 2] → result = [].concat([1, 2]) = [1, 2]
第2轮：arr = [3, 4] → result = [1, 2].concat([3, 4]) = [1, 2, 3, 4]
第3轮：arr = [5] → result = [1, 2, 3, 4].concat([5]) = [1, 2, 3, 4, 5]

返回 [1, 2, 3, 4, 5]

例子4：统计元素出现次数：

执行过程：

初始：obj = {}

第1轮：fruit = ‘apple’ → obj = { apple: 1 }
第2轮：fruit = ‘banana’ → obj = { apple: 1, banana: 1 }
第3轮：fruit = ‘apple’ → obj = { apple: 2, banana: 1 }
第4轮：fruit = ‘orange’ → obj = { apple: 2, banana: 1, orange: 1 }
第5轮：fruit = ‘banana’ → obj = { apple: 2, banana: 2, orange: 1 }
第6轮：fruit = ‘apple’ → obj = { apple: 3, banana: 2, orange: 1 }

返回 { apple: 3, banana: 2, orange: 1 }

总结
reduce 就是：

  • 从左到右遍历数组
  • 每次把当前元素和累加器按规则处理
  • 把处理结果作为新的累加器
  • 最后返回最终的累加器

可以用来做：求和、求积、找最大/小值、数组转对象、数据统计等等
